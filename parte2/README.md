## Requisitos

- Tener instalado Node (ejecutar `node -v` para verificar)
- Tener instalado Angular:

        npm i -g @angular/cli
        ng version

- Servir la API de la parte 2:

        cd api
        dotnet build
        dotnet run

- Instalar dependencias del front:

        cd web
        npm install

- Levantar el front:

        npm start
